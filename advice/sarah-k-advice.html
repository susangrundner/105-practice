Hey there Skillcrush peeps!

I'm Sarah and I have some advice for you!

Never, never, never give up!

Learning how to code is fun and exciting! And it's also REALLY diffcult!

No matter how hard it is to find the answer to whatever coding conundrum you're working on, don't give up! Use every resource available to you to find a solution. And when in doubt, ask a friend!

Have fun with it!

I get it, sometimes going through step by step lessons can be boring! But don't forget to have fun with the bonus and final projects!

I guarantee if you make the projects your own, you'll ending up learning more in the long run! You'll want to code up that awesome idea of yours to perfection if it's something you're excited about!
