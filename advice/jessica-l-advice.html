Hi, my name is Jessica and I have some advice for you!

Practice is the way

First of all, learning to code is hard stuff but it can be so fun to learn! Don't fret or start giving up if you don't get coding right away. That's why practice is your friend :)

Take things one step at a time. If you're learning and get stuck, don't be afraid to ask for help. You can also Google search for advice and tons of resources. Try do practice one thing until you master it before moving on to the next.

Discipline plays an important role

One thing that helped me remember concepts of coding is to do one problem or small assisgnment a day. This helps with retaining information so that you don't forget all the good things you've learned so far. This can also help you improve on your coding skills. Like the saying goes, practice makes perfect!
