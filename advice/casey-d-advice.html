Hi, my name is Casey and I have some advice for you!

Pretend it's a puzzle game

When you code your first single-page website, it might seem like everything you learned isn't quite working. If you look at it like a puzzle game, and you know that you have the key but just have to find it, it makes the whole process more enjoyable!

Be comfortable with trial and error (and Google)

When you first learn how to code a website, be comfortable with a lot of trial and error. Something that seemed really simple when working through an excercise may suddenly completely trip you up when you're coding on your own. Try one thing, try another, try the same thing again (because for some reason it might work this time), and keep trying. And when all else fails, don't be afraid to Google!
